# Automation Testing Project

This project demonstrates Selenium WebDriver automation using Java and TestNG.

## Tools & Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Framework Features
- Reusable Base Class
- Page Object Model Design Pattern
- TestNG Annotations
- Cross Browser Ready
- Maven Build Management

## Sample Test Covered
- Login Functionality
- Valid & Invalid Credentials
- Basic Assertion Validation
