## Project Description
This project demonstrates a basic automation framework developed using Selenium WebDriver and TestNG.

## Tools & Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Apache POI (Data Driven Testing)

## Framework Design
- Page Object Model (POM) design pattern used
- Separate folders for:
- Test Scripts
- Page Classes
- Utility Classes
- Test Data

## Features Implemented
- Login automation
- Fund transfer automation
- Data-driven testing using Excel
- cross-browser execution (Chrome)

## Reporting
- TestNG Reports generated after execution
